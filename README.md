# CP3402_prac1

## Lumina Fine Jewellery
## Project Description

This project is a modern jewelry e-commerce website designed to showcase and sell elegant jewelry collections online. The website focuses on providing customers with a luxurious and user-friendly shopping experience through clean layouts, high-quality product visuals, and responsive web design. It allows users to explore different jewelry categories, learn about the brand, and browse featured products seamlessly.

## Features

- Responsive Design
- Multi-Page Navigation
- Product Showcase
- Modern User Interface
- Brand Introduction Section
- Smooth scroll-triggered fade-up animations

## Technology Used

- HTML
- CSS
- Visual Studio Code
